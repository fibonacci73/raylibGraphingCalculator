# Graphing calculator made in c using raylib <br>

## Description<br>
This is a work in project graphing calculator made in c using [raylib](https://www.raylib.com/). It Can draw up to **7 function** and it accepts **sin, cos, tan, square roots, sums, subtractions, multiplications and divisions**.<br>
It also supports **variable scale** and it computes **intersections** between two functions. 

## Features<br>
- drawing up to seven functions
- using basic goniometric functions, the four operations and square roots
- changing scale of the graphic visualization
- calculating intersections between 2 functions

## Installation<br>
You can download the `game.exe` file only to execute it.<br>
To edit the code you gotta download everything in the `master` branch and then edit `makefile` inserting your path for raylib (ex. `RAYLIB_PATH        ?= C://raylib/raylib`)<br>
